Nettools 安装说明


本程序在x86_64 CPU运行，在Fedora Linux 40测试通过。

1、方式一：以 root 执行 dnf install http://www.nettools.top/linux/intel/fedora/nettools-20.20-1.fc42.x86_64.rpm
      方式二：下载 nettools-20.20-1.fc42.x86_64.rpm，然后以 root 执行 dnf install nettools-20.20-1.fc42.x86_64.rpm

2、执行 nettools  即可。


3、程序根据系统所使用的语言自动显示中文或者英文。如果你的计算机是英文，但希望程序使用中文，则在 home 目录下建立 .nettools 目录，并在 .nettools 目录中建立 nettools.conf 文件，文件内容为 LANGUAGE=Simplified Chinese ，重启程序即可。
