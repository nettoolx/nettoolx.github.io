DMG Installation Instructions


By default, MacOS does not allow the running and installation of apps downloaded from the internet, so settings need to be made to prevent alerts such as file corruption and inability to run apps.
Any of the following methods can be used.

Method 1: Open the "Any Source" option on MacOS
1. For MacOS 15 system, open System Settings - Privacy and Security - Security, select "Any Source";
2. For MacOS 14 and below systems:
1) Run sudo spctrl -- master disable, and then enter the password;
2) Open System Preferences - Security and Privacy - General - Allow apps to be downloaded from the following locations:, select 'Any Source'.
Then double-click the nettools.dmg file to install and run.

Method 2: Manually creating an APP
1. Double click nettools.dmg to load the nettools disk image;
In any directory (taking the home directory as an example), create the nettools. app directory:
mkdir ~/nettools.app
3. Copy the contents of the APP from the nettools disk image to this directory:
cp -r /Volumes/nettools/nettools.app/Contents ~/nettools.app
4. Double click on the app to run.
