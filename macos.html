DMG Installation Instructions


By default, MacOS computers do not allow running and installing apps downloaded from the internet, so you’ll need to adjust the settings; otherwise, warnings such as "File damaged" may appear, and the app won't run.
You can choose either of the following methods.

Method 1: Enable the “Allow from Anywhere” Option in MacOS
1. For MacOS 15: Go to System Settings > Privacy & Security > Security, and select “Allow from Anywhere.”
2. For MacOS 14 and below:
1) Run sudo spctrl -- master disable in the terminal, and then enter your password;
2) Go to System Preferences > Security & Privacy > Security, and select “Allow applications downloaded from: Anywhere.”
3. Open nettools.dmg, drag nettools to the Applications folder, and then double-click to run nettools.

Method 2: Manually Create the App
1. Double-click nettools.dmg to load the nettools disk image;
2. In any directory (using the home directory as an example), create a nettools.app folder:
mkdir ~/nettools.app
3. Copy the contents of the app from the nettools disk image to this directory:
cp -r /Volumes/nettools/nettools.app/Contents ~/nettools.app
4. Double-click the app to run it.
