Installation Instructions


This program runs on an x86_64 CPU and has passed testing on Fedora Linux 40.

1. Method 1: As root, run dnf install http://www.nettools.top/Linux/Intel/Fedora/nettools-20.0-1.fc40.x86_64.rpm
    Method 2: Download nettools-20.0-1.fc40.x86_64.rpm and run dnf install nettools-20.0-1.fc40.x86_64.rpm as root ;

2. Run nettools .
